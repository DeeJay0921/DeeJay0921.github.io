    关于字符串中的+

    对于字符串中的+操作会进行字符串的拼接（即调用默认的toString()），但是由于Java中的String不可变，所以JDK会偷偷使用StringBuider来进行字符串的拼接，从而避免了创建很多零碎的小String对象。
