Python中关于字典的一些API

clear()

pop(key) popitem()

update(dict2)

copy()

对于copy()的拷贝，该API返回一个字典的浅拷贝，关于如何深拷贝，以后再进行介绍。

values(),keys(),items()

对于这三个方法，如果是Python2的话返回的都是列表

fromkeys(iterable,[defaultValue])

setdefault(key,defaultValue)

setdefault()和get()类似，但是不同的地方在于，如果setdefault()方法没有找到指定的key，那么会相应的会创建一个key,其value为指定的第二参数，如果没指定即为None
